# Solar-Lander
a small [libGDX](https://libgdx.com/) orbit-sim project with a controllable lander 

![](assets/preview.png)
![](assets/preview2.png)

### Keybinds
```WASD``` : Cardinal Direction Movement \
```V (HOLD)``` : Show Forces Vectors of Lander \
```Space (HOLD)``` : Show Hitboxes \
```Z (HOLD)``` : Zoom Out
